### `var`

`var`는 **전역 변수**로 동작한다.
`var` 변수에 대해 브라우저와 node의 다음과 같은 차이가 있다.

- 브라우저 : 변수가 전역 객체에 포함
- Node : 변수가 지역 객체에 포함

### 동일명 변수에 재할당시 `var`와 `let`의 차이

선언 이후에 동일명 변수를 다른 스코프에서 재할당할 때 두 선언 키워드의 차이가 있다.

- `var` : 선언 키워드 사용 유무와 관계없이 값의 재할당이 이뤄진다.
- `let` : 선언 키워드를 사용하면 동일명의 새로운 변수가 생성된다. 사용하지 않으면 재할당이 이뤄진다.
